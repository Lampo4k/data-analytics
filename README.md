# Проект 1: Исследование надежности заемщиков

## Описание
Проект направлен на анализ данных о заемщиках банка для выявления факторов, влияющих на возврат кредита в срок. Исследование включает анализ зависимости между наличием детей, семейным положением, уровнем дохода и целями кредита. Данные содержат информацию о количестве детей, трудовом стаже, возрасте, образовании, доходе и цели кредита.

## Что я научилась
- Проводить предобработку данных: обрабатывать пропуски, аномалии и категоризировать данные.
- Анализировать зависимости между признаками и целевой переменной.
- Строить гистограммы, boxplot-ы и тепловые карты для визуализации данных.
- Делать выводы на основе статистического анализа, например, о влиянии семейного положения и дохода на возврат кредита.
- Использовать библиотеки pandas, numpy и seaborn для анализа и визуализации данных.
#Стек
- Python, pandas, numpy, matplotlib, seaborn, scipy.stats.

# Проект 2: Прогнозирование цен на аренду квартир

## Описание
Проект решает задачу прогнозирования цен на аренду квартир на основе данных с renthop.com. Используются признаки: количество ванных, спален, уровень интереса и цена. Цель — построить модель, предсказывающую стоимость аренды.

## Что я научилась
- Загружать и анализировать данные с Kaggle, используя pandas.
- Работать с пропусками и выбросами в данных.
- Создавать новые признаки на основе существующих, например, возводя их в квадрат.
- Разделять данные на обучающую и тестовую выборки.
- Обучать модели машинного обучения (линейная регрессия, дерево решений) и оценивать их качество с помощью метрик MAE и RMSE.
- Использовать библиотеки sklearn, matplotlib и seaborn для обучения моделей и визуализации результатов.

#Стек
- Python, pandas, numpy, sklearn, matplotlib, seaborn, lightgbm.
